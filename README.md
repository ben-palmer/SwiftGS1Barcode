# SwiftGS1Barcode
A GS1 Barcode Library and Parser for Swift Edit

## CocoaPod
https://github.com/xremix/SwiftGS1Barcode

## Deployment Steps
- Update Version
- Push Code to Git
- Create Release on Git
- `pod lib lint`
- `pod trunk push SwiftGS1Barcode.podspec`
